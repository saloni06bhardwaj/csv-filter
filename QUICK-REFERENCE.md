## 🎯 Common Tasks

### Upload & View
1. Drag CSV file to upload zone
2. Data loads automatically
3. View in table

### Filter Data
1. Type in global search OR
2. Click "Show Filters"
3. Type in specific column filter
4. Results update instantly

### Export Filtered Data
1. Apply filters
2. Hide unwanted columns (optional)
3. Click download icon
4. CSV downloads

### Navigate Pages
1. Use First/Previous/Next/Last buttons
2. View page info (Page X of Y)
3. 10 rows per page

### Scroll Wide Tables
1. Table wider than screen?
2. Scrollbar appears at bottom
3. Drag or Shift + wheel to scroll
4. All columns accessible

## 🐛 Troubleshooting

| Problem | Solution |
|---------|----------|
| No scroll | Table not wide enough or window too wide |
| Can't upload | Check file is .csv format |
| Filters not working | Clear all filters and try again |
| Export not working | Ensure data is loaded |
| Theme not changing | Click sun/moon icon in header |
