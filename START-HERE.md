## 📊 NEW: Visual Analytics!

### How to Use Charts
1. Upload CSV file
2. Click **📈 chart icon** in header
3. Select column from dropdown
4. Choose chart type: **Bar | Pie | Line**
5. Adjust "Show Top" (5, 10, 15, 20)
6. Done! Chart displays instantly

### Chart Types
- **Bar** - Compare categories
- **Pie** - Show proportions
- **Line** - Display trends

## 💡 Quick Examples

### Example 1: Department Chart
```
1. Upload sample-data.csv
2. Click chart icon
3. Select "Department"
4. Choose "Pie Chart"
→ See department distribution
```

### Example 2: Filtered Analysis
```
1. Upload CSV
2. Filter: Status = "Active"
3. Click chart icon
4. Select "Department"
→ See only active employees by dept
```

### Example 3: Export Filtered Data
```
1. Apply filters
2. Hide unwanted columns
3. Click download icon
→ Exports filtered CSV
```
