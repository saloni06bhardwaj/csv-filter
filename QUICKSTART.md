# 🚀 Quick Start Guide

## Start the Application

1. Open terminal/command prompt
2. Navigate to project folder:
   ```bash
   cd "d:\project - zift\ConsystemsAI task\project1"
   ```
3. Start the app:
   ```bash
   npm start
   ```
4. Browser opens automatically at `http://localhost:3000`

## Test with Sample Data

1. Use the included `sample-data.csv` file
2. Drag it onto the upload zone OR click "Choose File"
3. Explore all features!

## 💡 Pro Tips

1. **Quick Filter**: Use global search for fast results across all columns
2. **Precise Filter**: Use column filters for specific column searches
3. **Multi-Sort**: Click column headers to sort data
4. **Bulk Actions**: Select multiple rows and delete at once
5. **Custom View**: Hide unnecessary columns for cleaner view
6. **Export Filtered**: Only visible, filtered data gets exported

## 🎯 Common Use Cases

### Find Specific Records
1. Type in global search bar
2. Results update instantly

### Analyze by Column
1. Click "Show Filters"
2. Enter text in specific column filter
3. View filtered results

### Clean Up Data
1. Select unwanted rows
2. Click "Delete (X)"
3. Export cleaned data

### Focus on Key Columns
1. Click eye icon
2. Uncheck columns to hide
3. Cleaner, focused view
